# 人形机器人项目 - 阶段一 项目文档

## 📋 项目概述

这是一个基于语音控制的人形机器人系统，集成了语音识别、自然语言理解、计算机视觉、机器人控制等多个先进技术模块。项目采用模块化设计，支持实时语音交互、目标检测、精确抓取等功能。

### 🎯 核心功能
- **语音交互**：支持中文语音识别和语音合成
- **智能理解**：基于大语言模型的自然语言理解
- **视觉感知**：实时目标检测和位置定位
- **精确控制**：机器人手臂和手部的精确运动控制
- **动作执行**：预定义动作序列和自定义动作

## 📁 核心模块详解

### 1. 🎤 语音处理模块 (`voice/`)

#### 核心组件

##### `config.py` - 配置管理
- 集中管理所有配置参数
- API密钥配置（智谱AI、机器人连接等）
- 音频参数、机器人连接参数、唤醒词配置

##### `VAD.py` - 语音活动检测
- 实时检测语音开始和结束
- 支持多种VAD算法自动降级
- 实时音频缓冲和预处理

##### `ASR.py` - 语音识别
- 将语音转换为文字
- 基于SenseVoice/FunASR
- 支持多种音频格式，中文语音识别优化

##### `LLM.py` - 大语言模型接口
- 自然语言理解和指令解析
- 基于智谱AI GLM-4
- 机器人指令解析和上下文管理

##### `LLM_for_drink.py` - 饮料专用LLM接口
- 专门用于饮料相关指令处理
- 针对饮料识别和抓取优化
- 简化指令流程

##### `TTS.py` - 文本转语音
- 将文本转换为语音输出
- 基于Edge TTS
- 支持多语言，音频缓存机制

##### `PinyinMatcher.py` - 拼音匹配
- 唤醒词检测和命令识别
- 基于pypinyin的模糊匹配
- 拼音变体生成和模糊匹配算法

##### `ActionExecuter_hand.py` - 动作执行器
- 解析和执行机器人动作指令
- 动作队列管理和状态跟踪
- 与机器人控制模块集成

##### `utils.py` - 工具函数
- 通用工具函数集合
- 音频处理、文件操作、网络通信等
- 日志和调试工具

##### `deps.py` - 依赖管理
- 依赖检查和安装管理
- 模块可用性验证
- 错误诊断和安装建议

#### 语音模块工作流程
```
语音输入 → VAD检测 → ASR识别 → LLM理解 → 动作执行 → TTS反馈
```

## 🔧 技术架构

### 系统架构图
```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   语音输入模块   │    │   视觉感知模块   │    │   机器人控制模块  │
│   (VAD + ASR)   │    │ (目标检测+定位)  │    │  (手臂+手部)     │
└─────────┬───────┘    └─────────┬───────┘    └─────────┬───────┘
          │                      │                      │
          └──────────────────────┼──────────────────────┘
                                 │
                    ┌─────────────┴─────────────┐
                    │      中央控制模块          │
                    │   (VoiceRobotController)  │
                    └─────────────┬─────────────┘
                                  │
                    ┌─────────────┴─────────────┐
                    │       大语言模型接口       │
                    │        （LLM）            │
                    └───────────────────────────┘
```

### 数据流图
```
语音输入 → VAD检测 → ASR识别 → LLM理解 → 选择需要执行的动作序列 → 机器人执行
    ↓
视觉输入(若是需要抓取可乐) → 目标检测 → 位置计算(计算出抓取的饮料的序号) → 选择对应的动作序列 → 机器人执行
    ↓
动作执行结束输出语音 → TTS合成 → 语音播放
```

## 🚀 部署和使用

### 环境要求
- **Python版本**：3.10
- **操作系统**：Windows 10/11 或 Linux
- **硬件要求**：支持CUDA的GPU（推荐）

### 安装步骤

#### 1. 环境准备
```bash
conda create --name pr python=3.10
conda activate pr
pip install -r requirements.txt
```

#### 2. 模型下载
```bash
python -c "import torch; torch.hub.load('snakers4/silero-vad', 'silero_vad')"
export HF_ENDPOINT=https://hf-mirror.com
huggingface-cli download --resume-download hexgrad/Kokoro-82M-v1.1-zh --local-dir ./ckpts/kokoro-v1.1
```

#### 3. X5 API安装
```bash
cd whl
pip install xapi-3.1.11-cp310-cp310-win_amd64.whl  # Windows
pip install xapi-3.1.11-cp310-cp310-linux_x86_64.whl  # Linux
```

### 运行方式

#### 主程序运行
```bash
python main_hand.py  # 手部控制版本
python keyboardController.py # 键盘控制版本
```
