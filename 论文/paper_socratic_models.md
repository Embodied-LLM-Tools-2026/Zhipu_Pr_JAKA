## 核心思想速览

### 🎯 核心问题：零样本多模态推理

这篇论文的核心在于通过**多模态提示**组合多个预训练模型，实现零样本多模态推理。

**核心痛点**：
- 不同模型在不同领域训练，知识不重叠
- 难以组合多个模型的能力
- 缺乏零样本多模态推理能力

### ⚙️ 核心机制：多模态提示 → 模型组合

通过**语言作为中介**，让不同模型交换信息：

```
VLM → 文本描述 → LLM → 文本推理 → TTS → 语音输出
```

**关键创新**：
- **模块化组合**：多个预训练模型零样本组合
- **语言中介**：用语言作为模型间的通信协议
- **零样本能力**：无需联合训练即可工作

**示例**：
```
VLM："图像中有一个红色的苹果在桌子上"
LLM："用户问'我能吃这个苹果吗？' → 可以，苹果是可食用的"
TTS："是的，你可以吃这个苹果"
```

### 💡 核心意义

- ✅ **零样本组合**：无需联合训练
- ✅ **模块化**：可灵活替换模型
- ✅ **多模态能力**：视觉、语言、语音等
- ✅ **可扩展性**：易于添加新模型

### 📊 一句话总结

> **"用语言作为中介，零样本组合多个预训练模型"**

### 2.2 多模态提示

**提示设计**：
```python
# VLM 提示
vlm_prompt = "描述这个场景中的物体"

# LLM 提示
llm_prompt = f"""
场景描述：{vlm_output}
用户问题：{user_question}

请回答用户的问题。
"""
```

### 2.3 应用场景

**1. 视频问答**：
```python
def video_qa(video, question):
    # 1. 提取关键帧
    frames = extract_frames(video)
    
    # 2. VLM 描述每帧
    descriptions = [vlm.describe(frame) for frame in frames]
    
    # 3. LLM 回答问题
    prompt = f"""
    视频内容：{descriptions}
    问题：{question}
    
    请回答问题。
    """
    
    return llm.generate(prompt)
```

**2. 机器人规划**：
```python
def robot_planning(image, instruction):
    # 1. VLM 描述场景
    scene_desc = vlm.describe(image)
    
    # 2. LLM 生成计划
    prompt = f"""
    场景：{scene_desc}
    指令：{instruction}
    
    生成执行计划。
    """
    
    plan = llm.generate(prompt)
    
    return plan
```

## 五、可借鉴之处（针对我们的项目）

### 5.1 直接可借鉴

#### 借鉴 1：模型组合

**应用到我们的项目**：

```python
# voice/agents/model_composer.py

class SocraticComposer:
    """Socratic 模型组合器"""
    
    def __init__(self):
        self.vlm = VLMModel()
        self.llm = LLMModel()
        self.tts = TTSModel()
    
    def compose(self, image, instruction):
        # 1. VLM 描述场景
        scene_desc = self.vlm.describe(image, "描述场景中的物体")
        
        # 2. LLM 生成计划
        prompt = f"""
        场景：{scene_desc}
        指令：{instruction}
        
        生成执行计划。
        """
        plan = self.llm.generate(prompt)
        
        return plan
```

#### 借鉴 2：多模态提示

**设计多模态提示**：

```python
def build_multimodal_prompt(vlm_output, user_input, context):
    """构建多模态提示"""
    prompt = f"""
    # 视觉感知
    {vlm_output}
    
    # 用户输入
    {user_input}
    
    # 上下文
    {context}
    
    请综合以上信息，生成回答或计划。
    """
    
    return prompt
```
