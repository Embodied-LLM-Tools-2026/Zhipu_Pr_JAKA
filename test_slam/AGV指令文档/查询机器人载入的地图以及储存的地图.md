**查询机器人载入的地图以及储存的地图**

**请求**

编号: 1300 (0x0514)

名称: robot_status_map_req

描述: 查询机器人载入的地图以及储存的地图

JSON 数据区: 无

**请求示例**

Plaintext  
5A 01 00 01 00 00 00 00 05 14 00 00 00 00 00 00

**响应**

编号: 11300 (0x2C24)

名称: robot_status_map_res

描述: 查询机器人载入的地图以及储存的地图的响应

JSON 数据区: 见下表

|     |     |     |     |
| --- | --- | --- | --- |
| **字段名** | **类型** | **描述** | **可缺省** |
| current_map | string | 当前载入的地图 | 是   |
| current_map_md5 | string | 当前载入地图的md5值 | 是   |
| maps | array\[string\] | 所有储存在机器人上的地图名组成的数组 | 是   |
| map_files_info | array\[json\] | 地图信息数组，包括地图名，地图大小，最后修改时间 | 是(3.4.6增加） |
| ret_code | number | API 错误码 | 是   |
| create_on | string | API 上传时间戳 | 是   |
| err_msg | string | 错误信息 | 是   |
