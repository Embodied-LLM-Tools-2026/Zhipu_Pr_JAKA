# Voice 语音机器人模块

这个文件夹包含了语音机器人系统的所有核心模块，采用模块化设计，每个文件负责特定的功能。

## 📁 文件结构

### 🔧 核心配置文件

#### `config.py` (2.6KB, 88行)
- **功能**: 集中管理所有配置参数和环境变量
- **包含内容**:
  - API密钥配置 (ZhipuAI)
  - 机器人连接参数 (IP, 端口)
  - 音频参数 (采样率, 声道数, 块大小)
  - 缓存和预加载设置
  - 常用短语和动作映射
  - 唤醒词和取消词
- **使用方式**: 通过 `from voice.config import *` 导入配置

### 🛠️ 工具和依赖

#### `deps.py` (8.1KB, 225行)
- **功能**: 依赖管理和检查
- **主要类**: `DependencyManager`
- **功能**:
  - 检查所有必需依赖
  - 验证模块可用性
  - 提供安装建议
  - 错误诊断

#### `utils.py` (50KB, 1196行)
- **功能**: 通用工具函数
- **包含内容**:
  - 音频处理工具
  - 文件操作工具
  - 网络通信工具
  - 日志和调试工具
  - 性能监控工具

### 🔊 语音活动检测模块

用来自动识别语音开始和结束

#### `VAD.py` (30KB, 727行)
- **功能**: 语音活动检测 (Voice Activity Detection)
- **技术**: 支持多种VAD算法
- **主要类**:
  - `VoiceActivityDetector`: 基础能量检测
  - `SileroVAD`: Silero VAD模型
  - `CircularBuffer`: 音频缓冲
  - `AdvancedVoiceActivityDetector`: 高级VAD
- **功能**:
  - 实时检测语音活动
  - 支持多种VAD算法
  - 音频缓冲和预处理
  - 自动降级机制

### 🎯 唤醒模块

#### `PinyinMatcher.py` (5.5KB, 144行)
- **功能**: 拼音模糊匹配
- **技术**: 基于pypinyin
- **主要类**: `PinyinMatcher`
- **功能**:
  - 唤醒词检测
  - 取消命令检测
  - 拼音变体生成
  - 模糊匹配算法

### 🎤 语音处理模块

#### `ASR.py` (1.4KB, 41行)
- 输入：语音；输出：文字
- **功能**: 语音识别 (Automatic Speech Recognition)
- **技术**: 基于SenseVoice/FunASR的语音转文字
- **主要类**: `SenseVoiceRecognizer`
- **功能**:
  - 加载语音识别模型
  - 实时语音转文字
  - 支持多种音频格式

#### `LLM.py` (6.5KB, 138行)
- 输入：文字；输出：大模型的正常对话回答或者动作指令
- **功能**: 大语言模型接口
- **技术**: 基于ZhipuAI GLM-4
- **主要类**: `LLMInterface`
- **功能**:
  - 自然语言理解
  - 对话生成
  - 命令解析
  - 上下文管理

#### `TTS.py` (4.6KB, 110行)
- 输入：文本；输出：语音
- **功能**: 文本转语音 (Text-to-Speech)
- **技术**: 基于Edge TTS
- **主要类**:
  - `TextToSpeechEngine`: TTS引擎
  - `StreamingTTSPlayer`: 流式播放器
- **功能**:
  - 文本转语音
  - 音频缓存
  - 流式播放
  - 多语言支持

### 🤖 机器人控制模块

#### `ActionExecuter.py` (2.4KB, 70行)
- 输入：LLM提供的动作指令；输出：去调用X5或者其他的控制程序来执行动作
- **功能**: 动作执行器
- **主要类**: `ActionExecuter`
- **功能**:
  - 解析和执行机器人动作
  - 动作队列管理
  - 动作状态跟踪


### 🧠 主控制器

#### `VoiceRobotController.py` (39KB, 983行)
- **功能**: 语音机器人主控制器
- **主要类**: `VoiceRobotController`
- **功能**:
  - 整合所有模块
  - 状态管理 (睡眠/唤醒)
  - 命令处理流程
  - 性能监控
  - 错误处理


### 📝 原始文件

#### `origin_main.py` (150KB, 3714行)
- **功能**: 原始的单文件版本
- **说明**: 这是模块化之前的完整实现，保留作为参考
